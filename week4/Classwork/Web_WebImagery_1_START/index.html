Web Image Types

Jpg (Joint Photographic Experts Group)

  1. Millions Colors
  2. Photographs (continuous, highlights shadows, and everything in between)
  3. Proprietary Compression

Gif (Graphics Interchange Format)

  • 256 Colors
  • Flat Graphics
  • Transparency
  • Animation (animated gif)

Png-8 (Portable Network Graphics)

  • 256 Colors
  • Flat Graphics
  • Transparency
  • No Animation

Png-24 (Portable Network Graphics)

  1. Millions Colors
  2. Photographs (continuous, highlights shadows, and everything in between)
  3. No Proprietary Compression
  4. True Transparency
  5. *Issues with older browsers

SVG (Scalable Vector Graphics)

  1. Millions Colors
  2. Vector image format (infinitely scalable)
  3. No Proprietary Compression
  4. True Transparency
  5. *Issues with older browsers
