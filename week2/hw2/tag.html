Tag Selector

A Tag selector is used to apply styles to any HTML tag within an HTML document (e.g. p, h1, a, b, div, body). Because a tag selector applies to all instances of the tag selected, it is the least specific selector. Classes and ID's are used to give the author more control over specific elements that need to stand apart visually.
